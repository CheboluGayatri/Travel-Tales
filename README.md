# 🌍 Travel-Tales

A simple **HTML & CSS only** project that showcases a travel-themed website called **Travel-Tales**.  
This project is designed to practice front-end basics, including layout, styling, and responsive design.

---

## ✨ Features
- 🏝️ Beautifully styled travel landing page  
- 📖 About page to describe the website/brand  
- 📞 Contact page with a simple form layout  
- 🎨 Responsive design using only **HTML5 & CSS3** (no JavaScript)
