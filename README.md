# Travel Tales 🌍✈️

**Travel Tales** is a beautifully designed travel website that showcases some of India's most enchanting destinations — from the high passes of Leh-Ladakh to the spiritual heart of Varanasi and the lush green Western Ghats.

## 🌟 Project Overview

Travel Tales is a front-end web project built using **HTML5** and **CSS3**, designed to be fully **responsive** on both desktop and mobile devices. It allows users to explore brief highlights about each location, understand why each place is worth visiting, and get in touch via a contact form.

## 🧭 Features

- 🗺️ Clean layout with top destinations and descriptions
- 📱 Fully mobile-responsive design
- 🖼️ Image cards with hover effects
- 💬 Contact form with modern input styling
- 🔗 Smooth scroll navigation
- ⚡ Fast, lightweight, and easy to customize

## 🔧 Tech Stack

- **HTML5**
- **CSS3**
